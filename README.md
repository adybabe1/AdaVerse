## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/adaverse/company-website.git
   cd company-website
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open the browser and navigate to `http://localhost:3000`
